# 1С Trusted Gateway (Go)

Лёгкое Windows-приложение для безопасного доступа AI-агентов к данным 1С через MCP-протокол.

**Вес:** ~11 МБ · **Запуск:** мгновенный · **Зависимости:** нет

---

## Что это

Trusted Gateway — посредник между AI-агентом и вашей базой 1С:

```
AI-агент  ──TCP bridge──▶  Trusted Gateway  ──MCP/HTTP──▶  1С
                                   │
                            маскирует ФИО,
                            серийники, коды
```

Агент **никогда не видит** реальные имена, ИНН, КПП, серийные номера. Вместо этого он получает псевдонимы (`Контрагент_a3f2b1c0d8`). Суммы и количества передаются открыто — агент может анализировать цифры, но не деанонимизировать людей.

Текст анализа от агента приложение расшифровывает локально — пользователь видит реальные имена, агент — нет.

## Требования

- Windows 10/11 (x64)
- 1С:Предприятие 8.3 с установленным расширением **MCP_Сервер.cfe** из папки [1c-extension/](1c-extension/)

Для сборки из исходников дополнительно нужен [Go 1.21+](https://go.dev/dl/).

## Быстрый старт

1. Скачайте последний релиз со страницы [Releases](https://github.com/alonehobo/1c-trusted-gateway/releases) — там есть `TrustedGateway.exe` и `MCP_Сервер.cfe`
2. Установите расширение `MCP_Сервер.cfe` в вашу базу 1С
3. Убедитесь, что HTTP-сервис `hs/mcp` опубликован и доступен
4. Запустите `TrustedGateway.exe`
5. Откроется браузер с интерфейсом на `http://127.0.0.1:8767`
6. Введите URL вашего MCP-сервера и токен доступа, нажмите **Подключиться**
7. Передайте агенту **Bridge Secret** из верхней панели

## TCP Bridge API

Агент подключается через TCP на `127.0.0.1:8766`. Протокол: JSON-запрос → `shutdown(SHUT_WR)` → JSON-ответ.

```python
import socket, json

def bridge(command, secret, **kwargs):
    payload = {"command": command, "secret": secret, **kwargs}
    with socket.create_connection(("127.0.0.1", 8766), timeout=300) as s:
        s.sendall(json.dumps(payload, ensure_ascii=False).encode("utf-8"))
        s.shutdown(socket.SHUT_WR)
        chunks = []
        while True:
            c = s.recv(65536)
            if not c: break
            chunks.append(c)
    return json.loads(b"".join(chunks).decode("utf-8"))
```

## Маскировка данных

| Тип значения | Действие |
|-------------|----------|
| Пустые значения, `null` | Не маскируются |
| Числа Go (`int`, `float64`, `bool`) | Не маскируются |
| Числовые строки ≤10 знаков без ведущих нулей | Не маскируются (суммы) |
| Числовые строки с ведущими нулями (`00000142`) | Маскируются (серийники) |
| Числовые строки >10 знаков | Маскируются (длинные коды) |
| ИНН, КПП (по имени поля) | Всегда маскируются |
| Все остальные строки | Маскируются |
| `force_mask` поля | Всегда маскируются |
| `allow_plain` поля | Никогда не маскируются |

Псевдонимы детерминированы: одно значение всегда даёт один и тот же псевдоним в рамках сессии (соль = HMAC от серверного токена).

## Безопасность

- **Настройки** (URL, токен) зашифрованы через Windows **DPAPI** — только ваш пользователь может расшифровать
- **Bridge Secret** генерируется заново при каждом запуске (32 случайных байта)
- **Маскировка принудительна** — bridge всегда возвращает маскированный режим, агент не может запросить иное
- **Только localhost** — HTTP-сервер и TCP-bridge слушают только `127.0.0.1`
- **Таймаут запросов** — 300 секунд, лимит строк 5 000, лимит размера ответа 10 МБ
