# Архив памяти проекта

Сюда можно переносить завершенные или устаревшие артефакты памяти, которые уже не нужны в рабочем контексте, но могут пригодиться для истории.

Примеры:

- старые change-папки после закрытия больших инициатив;
- временные заметки, превращенные в устойчивые паттерны или решения;
- устаревшие спецификации, которые важно сохранить для аудита.
