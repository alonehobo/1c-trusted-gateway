# UX Improvements — Spec Delta

## Что меняется в поведении

### Новые UI-элементы

| Элемент | Где появляется | Условие показа |
|---------|---------------|----------------|
| Welcome overlay | Поверх `#app` | Нет подключения + нет флага в localStorage |
| Кнопка "История" | Pane header редактора | Всегда |
| Dropdown истории | Под кнопкой "История" | По клику |
| Кнопка "CSV" | `#result-actions` | Есть результат |
| Контекстное меню | Над `#result-table` | Правый клик на ячейке |
| Splitter | Между панелями запрос/ответ | Всегда (вкладка MCP) |
| Toggle темы | Вместо двух кнопок в header | Всегда |
| Кнопки A-/A+ | `#result-actions` | Всегда |
| Elapsed timer | В badge `#query-state` | `query_state === "running"` |
| Кнопка "Сохранить запрос" | Pane header редактора | Всегда |

### Новые горячие клавиши

| Комбинация | Действие | Контекст |
|-----------|----------|----------|
| `Ctrl+Enter` | Выполнить запрос / одобрить код | Фокус в Ace Editor |
| `Escape` | Отменить запрос | Глобально |
| `Ctrl+S` | Сохранить настройки | Вкладка Settings |
| `Ctrl+Shift+H` | Открыть историю | Глобально |

### Новые данные в localStorage

| Ключ | Тип | Описание |
|------|-----|----------|
| `onboarding_done` | boolean | Онбординг пройден |
| `query_history` | JSON array | История запросов (max 50) |
| `saved_queries` | JSON array | Сохранённые запросы |
| `splitter_ratio` | number | Пропорция сплиттера (0.0–1.0) |
| `table_font_size` | number | Масштаб таблицы (px) |
| `sound_notifications` | boolean | Звуковые уведомления |

### Новые API endpoints (опционально, для UX-05 этап 2)

| Endpoint | Метод | Описание |
|----------|-------|----------|
| `GET /api/metadata_hints` | GET | Возвращает имена таблиц и полей для автодополнения |

### Что НЕ меняется

- Протокол MCP (JSON-RPC 2.0)
- Логика маскировки / NER
- Структура `appState` (серверная часть)
- API endpoints: `/api/state`, `/api/rows`, `/api/query`, `/api/settings`
- Формат конфигурации `config.json`
- Поведение авто/ручного режима
- SSE-подписка
